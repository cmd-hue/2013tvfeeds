# 2013tvfeeds
feeds for https://cmd-hue.github.io/2013tv
# Template for adding custom feeds
note: you must put the code inside _callbacks\_._smash( ); or window._callbacks\_._smash( );
# How to use
## Step 1
go to https://cmd-hue.github.io/2013tvfeeds/template. it should possibly download a file.
## step 2
edit the file to your liking.
## step 3
join the cmd hue discord https://discord.gg/c3SkVn4naY and go to #feed-submissions and upload your edited file. or open a push request on github by clicking [here](https://cmd-hue.github.io/2013tv/feedpush.html).
